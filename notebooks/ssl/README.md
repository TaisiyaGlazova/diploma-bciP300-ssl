# SSL notebooks

Ноутбуки для self-supervised pretraining.

## Основные ноутбуки

* `ssl_pretraining.ipynb`
  — обучение модели на задаче masked reconstruction

* `ssl_results_plots.ipynb`
  — анализ результатов SSL-предобучения:

  * кривые функции потерь
  * примеры реконструкции сигналов
  * визуализация качества обучения

## Результаты

* чекпоинты и логи сохраняются в `outputs/ssl/`
* описание эксперимента: `docs/ssl_pretraining.md`
